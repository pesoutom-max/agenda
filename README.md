# FacilPyme Agenda 📅

Prototipo de agenda automática para profesionales de salud independientes.

## 🚀 Despliegue en la Web

Este proyecto está configurado para desplegarse automáticamente en **GitHub Pages**.

### Cómo habilitarlo:
1. Ve a la pestaña **Settings** de este repositorio en GitHub.
2. En el menú de la izquierda, selecciona **Pages**.
3. En la sección **Build and deployment**, cambia la "Source" a **GitHub Actions**.
4. ¡Listo! En unos segundos tu sitio estará disponible en la URL que te proporcionará GitHub.

## 🛠️ Tecnologías
- **Frontend**: HTML5, CSS3 vanilla, JavaScript ES6.
- **Base de Datos**: Firebase Firestore.
- **Despliegue**: GitHub Actions + GitHub Pages.

## 👨‍⚕️ Vistas
- **Paciente**: `index.html` (Vista principal de reserva).
- **Profesional**: `pro.html` (Dashboard de citas de hoy).
- **Administración**: `admin.html` (Gestión de bloqueos y ediciones).
